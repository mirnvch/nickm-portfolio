Personal portfolio website with professional/personal mode toggle.

**Live:** [nickm-portfolio.vercel.app](https://nickm-portfolio.vercel.app)

## Tech Stack

- **Framework:** Astro
- **Language:** TypeScript (strict mode)
- **Styling:** TailwindCSS
- **Deployment:** Vercel

## Pages

| Route | Description |
|---|---|
| `/` | Homepage with about, projects, posts |
| `/blog` | Blog listing |
| `/blog/[slug]` | Blog post |

## Features

- Dark/light mode toggle (professional vs personal)
- View Transitions API for page navigation
- Responsive design

## Getting Started

```bash
npm install
npm run dev
```

Open [localhost:4321](http://localhost:4321).
